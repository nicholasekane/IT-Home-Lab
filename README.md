# IT Support Home Lab Portfolio

## Overview
This repository contains a structured IT support home lab built to practice and document common service desk and desktop support tasks. The focus is on developing hands-on experience with core enterprise tools and workflows in a controlled, non-production environment.

## Purpose of This Lab
The purpose of this lab is to build practical, entry-level IT support skills through direct, hands-on practice. Each lab is designed to reflect real-world support tasks such as user management, system configuration, and basic troubleshooting.

All work in this repository is performed in a home lab or sandbox environment and does not represent production experience.

## Tools and Technologies

## Lab Structure
The repository is organized by topic to reflect different areas of IT support:

- `01-active-directory/` — User and group management, account tasks  
- `02-windows-client/` — Domain join, login testing, client-side tasks  
- `03-microsoft-365/` — Admin concepts and user management  
- `04-servicenow/` — Ticket handling and workflow practice  
- `05-okta-optional/` — Identity and access management basics (optional)  
- `screenshots/` — Supporting images for lab documentation  
- `templates/` — Reusable documentation templates
